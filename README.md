# KanalMSG - secure messenger
КаналMSG - защищённый свободный российский мессенджер. Разработка ведётся с 21 февраля 2026 года. Релиз мессенджера планируется в августе-сентябре этого года.

Мессенджер написан на Python 3.14, для UI используется библиотека customtkinter.

Выложен пока только клиент для Windows.
В планах - версии для Android, Linux, iOS, браузерная версия. Большинство из них будут готовы к релизу.

Безопасность и конфиденциальность:
1. Все переписки шифруются E2EE
2. В качестве алгоритма шифрования используется AES-GCM
3. DH(2048 бит сейчас, позже планирую увеличить до 4096). Реализован с помощью библиотеки cryptography.
4. Защита от брутфорса паролей(на сервере, его исходный код пока закрыт). Блокировка по IP, потом и глобальная временная блокировка входа в аккаунт, если атака продолжается.
5. Используются Pairwise Keys, если в комнате больше 2-х пользователей

КаналMSG комнатно-ориентированный. К сожалению, на данный момент чат только текстовый.
К релизу будет добавлена возможность отправки изображений в форматах .jpeg, .jpg, .png. На этом проект останавливаться не будет.

Используется собственный сетевой протокол - SKTP(Secure Kanal Transport Protocol). Он сделан поверх TCP/IP.
